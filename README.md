# MoonLamp_8266
RGB Color Picker web server with ESP8266-01 (ESP-01)
-
#### Wiring Circuit  :: Resistor values for Moon Lamp.
  
0=Red (GPIO0) D3   :: 100R

2=Green (GPIO2) D4  :: 480R

3=Blue (GPIO3) Rx   :: 220R
